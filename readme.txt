oneview Mitglieder und Netzwerke Trendwolken mit Suchformular

== Description ==

Das oneview Trendwolken-Widget ermöglicht dir deine eigene Trendwolke oder Trendwolken von Netzwerken in denen du Mitglied bist in deinem Wordpress-Blog zu integrieren. Damit haben deine Leser immer Zugriff auf die von dir oder von anderen Mitgliedern deiner Netzwerke empfohlenen Webseiten, News, Videos. Das Trendwolken-Widget ist in Größe, Tag-Anzahl und Layout über den oneview Widget-Konfigurator frei gestaltbar. Das Trendwolken-Widget wird inklusive Suchformular für deine öffentlichen Links oder die eines Netzwerkes, sowie Kontakt-Informationen ausgeliefert.


== Installation ==

1. Entpacke die Datei oneviewWidget.zip oder oneview-widget.zip
2. Lade die Datei `oneviewWidget` oder `oneview-widget` in das Verzeichnis `/wp-content/plugins/` auf dem WordPress Server  hoch
3. Aktiviere das hochgespielte oneview plugin im WordPress Menue 'Plugins' 
4. Gehe zu Presentation ->  Widgets (/wp-admin/widgets.php) und ziehe per drag & drop das oneview Widget von den dort angebotenen Widgets in die Sidebar
5. Klicke auf "Configure"( icon rechts neben der Überschrift (oneview Widget) ) und gestalte und konfigurier es dir so wie du es möchtest

== Frequently Asked Questions ==

= Was ist oneview? =

oneview ist eine menschliche Suchmaschine und dein Online-Gedächtnis.
Entdecke die neuesten Nachrichten und die besten Webseiten mit den Augen von vielen. Nichts ist spannender als das, was gerade andere bewegt und im Trend ist. Bist du nicht neugierig?
Und das Beste ist, oneview ist gleichzeitig dein Online-Gedächtnis. Du kannst dir mit oneview alles aus der digitalen Welt merken (online bookmarking): Videos, News, Blog-Artikel, Webseiten, etc., alles was dir wichtig ist oder du einfach nicht vergessen möchtest.

Nutze oneview mit deinen Freunden zusammen. Jeder deiner Freunde ist ein weiterer Informations-Scout, dessen öffentliche Bookmarks sich zu deinen gesellen. Das Ergebnis ist die perfekte Empfehlungssuchmaschine, das Wissen deines Freundeskreises in deiner ganz persönlichen Suchmaschine.
Nutze es völlig privat, wie ein Bankschliessfach, oder öffentlich und profitier von mit anderen geteiltem Wissen. Gründe Themen-Netzwerke und tausche dich mit anderen Mitgliedern aus.
